# AgroStock AI

Sistema Inteligente de Gestión de Inventario Agropecuario desarrollado como proyecto final de la carrera Tecnólogo en Desarrollo de Software.

## Descripción

AgroStock AI es una plataforma web que permite administrar productos agropecuarios, controlar inventarios, registrar movimientos de entrada y salida, generar alertas de stock bajo y visualizar estadísticas mediante un dashboard.
